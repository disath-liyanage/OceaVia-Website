Home Page validation report

After running the pages I implemented through an HTML validator, I noticed a few syntax issues—specifically the use of self-closing slashes (/) in tags like <meta>, <link>, and <img>, which are not required in HTML5. The validator also highlighted an incorrect MIME type images/png, which I corrected to image/png. Based on this feedback, I updated the code to align with HTML5 standards.

Table Page validation report

When validating the HTML code for the Table page, the only issue highlighted was the unnecessary use of self-closing slashes (/) on <img> tags. Since the project uses HTML5, these slashes are not required and can be safely removed. After identifying this through the validator, I updated the <img> elements by removing the / at the end. This brought the code in line with HTML5 standards.
